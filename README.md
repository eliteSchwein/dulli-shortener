### Dulli Shortener

Dulli Shortener is a super basic url shortener that is written in symfony/php and doesnt collect any data and doesnt support any apps

### Database Commands
```shell
# migrate current database
php bin/console doctrine:migrations:migrate

# make a new database migration
php bin/console make:migration
```
